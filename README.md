# Dino Memory Game

A kid-friendly memory card game featuring dinosaurs. Built with TypeScript and Vite.

## Features

- 8 cards with 4 dinosaur pairs (T-Rex, Sauropod, Triceratops, Pterodactyl)
- Smooth card flip animations
- Match detection with visual feedback
- Win celebration message
- Fully responsive (mobile, tablet, desktop)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
```

Output in `dist/` folder.

## Tech Stack

- TypeScript
- Vite
- CSS Grid + Flexbox
- CSS Animations

## Deploy

Configured for Netlify deployment. Push to connected repo or drag `dist/` folder to Netlify.
